# Parodontose - Medizinische Fortbildung

## Übersicht

Interaktive Quiz-Website für das Erlernen von Parodontitis (Parodontose), konzipiert für Zahnmedizinstudentinnen und -studenten sowie zahnmedizinische Fachkräfte. Diese Plattform bietet strukturierte, evidenzbasierte Inhalte zur Vertiefung des Wissens in Klinik und Praxis.

## Lernmodule

Die Website umfasst **4 spezialisierte Module** mit insgesamt **100 Fragen**:

### 1. Ärztliche Untersuchung
Fundamente der klinischen Diagnostik:
- PSI (Periodontal Screening Index)
- Sondierungstiefe und Blutung
- Röntgenologische Befunde
- Mundhygieneassessment

### 2. Differentielle Diagnostik
Abgrenzung und Klassifikation:
- Differenzierung von Gingivitis und Parodontitis
- 2018er EFP/AAP-Klassifikation
- Systemische Erkrankungen und Parodontitis
- Risikoaktion und Modifizierungsfaktoren

### 3. Behandlung
Therapiekonzepte und -verfahren:
- Subgingivale Instrumentierung (SRP)
- Chirurgische Verfahren
- Unterstützende Parodontaltherapie (UPT)
- Rezidivprävention und Compliance

### 4. Formen und Klassifikation
Staging und Grading der Erkrankung:
- Klassifikationssysteme
- Schweregraduierung
- Komplexitätsbewertung
- Prognostische Faktoren

## Verwendung

### Lokale Nutzung
Öffnen Sie einfach die `index.html` im Webbrowser Ihrer Wahl:
```
Datei öffnen → index.html
```

Kein Server oder Installation erforderlich.

### Bereitstellung online
Die Website ist GitHub Pages-kompatibel und kann kostenfrei auf GitHub gehostet werden:
1. Repository zu GitHub pushen
2. GitHub Pages in Einstellungen aktivieren
3. Website wird automatisch unter `[username].github.io/[repo-name]` bereitgestellt

## Technologie

- **Frontend**: Statisches HTML, CSS und JavaScript
- **Mathematik**: KaTeX für medizinische Formeln und Notationen
- **Kompatibilität**: Alle modernen Browser (Chrome, Firefox, Safari, Edge)
- **Deployment**: GitHub Pages-optimiert, keine Backend-Anforderungen
